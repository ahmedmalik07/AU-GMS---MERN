# FGMS Backend (Air University Central Gym Management System)

## Features
- User authentication (JWT)
- Member management
- Attendance tracking
- Membership management
- Reports

## Tech Stack
- Node.js, Express
- MongoDB (Mongoose)
- JWT, bcryptjs
- CORS, dotenv

## Getting Started
1. Install dependencies:
   ```powershell
   npm install
   ```
2. Set up your `.env` (already provided for local MongoDB).
3. Start the server:
   ```powershell
   node index.js
   ```
